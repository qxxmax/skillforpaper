# Slide Quality Gate Checklist

Project:
Deck version:
Date:

## Story

- [ ] Author-approved or explicitly inferred framework exists.
- [ ] Every slide has one main claim.
- [ ] Every section has a clear transition.
- [ ] Talk timing matches slide count.
- [ ] Summary restates the thesis without overclaiming.

## Source Grounding

- [ ] Claim/evidence ledger updated or explicitly marked unchanged.
- [ ] Visual source ledger complete for all non-original visuals.
- [ ] All citations verified.
- [ ] Unverified sources are excluded from final deck or moved to TODO/backup.
- [ ] Copyright-sensitive images handled by citation, redraw, or permission policy.

## Formula And Notation

- [ ] Equation/notation audit complete.
- [ ] All formulas compile.
- [ ] Symbols are introduced before use.
- [ ] Units are consistent.
- [ ] Signs, factors, and exponents checked.
- [ ] Long formulas are readable or moved to backup.

## Visual Design

- [ ] Style profile applied consistently.
- [ ] Text is readable at presentation distance.
- [ ] Color use is consistent and accessible.
- [ ] Figures have message captions.
- [ ] Tables are not overloaded.
- [ ] Backup slides are clearly marked.

## Typo And Export

- [ ] Spelling/typo pass complete.
- [ ] Page numbers and section titles checked.
- [ ] PDF export checked visually.
- [ ] PPTX/TeX/Quarto source builds from a clean checkout or clean copy.
- [ ] Final artifacts linked in the final response or revision state.
