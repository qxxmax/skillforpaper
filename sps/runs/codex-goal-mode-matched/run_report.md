# Run Report

The complete SPS literature protocol was executed in Codex goal mode in an
isolated output directory. The run confirmed the root identity, screened all 58
live references, executed 30 fresh and 6 fixed legacy routes, deduplicated 578
candidates, read 27 primary PDFs across four dimensions, performed two closure
rounds, and generated the evidence, lineage, gap, and report artifacts.

The initial scripted database pass was blocked by sandbox DNS. That failed pass
is retained in `network_access_log.csv`; the authorized rerun completed all 36
routes successfully. Exact model and token/cache counters were not exposed and
were not estimated.

See `literature_research_report.md` for the scientific synthesis,
`coverage_stopping_report.md` for bounded stopping logic,
`final_validation_report.md` for the independent gates, and
`../../comparison/cost_effect_summary.md` for the curated public comparison.
