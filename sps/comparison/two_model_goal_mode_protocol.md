# Pending controlled two-model goal-mode experiment

Status: designed, not yet executed.

The current artifacts contain one fully identified model assignment and one
goal-mode runtime whose exact deployment identifier was not exposed. A strict
two-model result therefore requires a new controlled experiment.

## Fixed contract

1. Use the same SPS root identity and frozen 58-reference bibliography.
2. Use the same 30 fresh routes and 6 fixed legacy routes.
3. Require two closure rounds and the same four reading dimensions.
4. Start each arm in a clean output directory with no access to prior results.
5. Assign and record an explicit model identifier before each goal starts.
6. Record the goal counter at start, freeze, and completion.
7. Run at least three repetitions per model.

## Reported cost

- goal-level token delta;
- wall-clock time;
- tool and network calls;
- blocked or retried operations;
- output bytes and file count;
- API cost only when model-specific usage categories and prices are observable.

## Reported effect

- root-reference recovery;
- candidate and source-verified counts;
- verified pages and reading records;
- claim-source resolution rate;
- numerical-claim anchoring rate;
- checked lineage edges;
- unresolved gaps;
- final validation status.

No model-quality or price claim should be made until both model identifiers and
the measurement boundary are observable under this matched protocol.
