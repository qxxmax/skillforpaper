# Public export boundary

Included:

- the complete skill definition, references, templates, and scripts;
- SPS invocation and runtime manifests;
- search, screening, reading, evidence, numerical, lineage, and gap ledgers;
- generated lineage, landscape, and audit-funnel views;
- selected source screenshots used for presentation and checking;
- validation reports and the audit workbook.

Excluded:

- unrelated research examples and slide projects;
- downloaded third-party paper PDFs;
- long source extracts, full-text caches, and arXiv source archives;
- raw API metadata payloads;
- large inspection/debug intermediates;
- credentials and environment files.

The public package keeps canonical source links so a reviewer can return to the
original source without redistributing the source corpus.
